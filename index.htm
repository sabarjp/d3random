Diablo 3 v1.03 Random Build Generator

Use this tool to generate random builds for use in Diablo 3. You can try to confine to the build to a farming run, a difficulty level, or from character creation to death (yes, you have to auto-attack until you get the skill and rune!). For extra fun, try hardcore only using a random build. Note that some builds may be totally unviable. It may not be possible to assign skills to mouse clicks depending on what the skill does.
